## API Documentation

### Refresh Control Methods

#### `handleIntervalChange()`
- Triggered when interval dropdown changes
- Updates refresh interval dynamically
- Saves preference to localStorage

#### `handleRefreshToggle()`
- Pauses or resumes auto-refresh
- Toggles button state and countdown display

#### `handleManualRefresh()`
- Triggers immediate data fetch
- Bypasses current auto-refresh state
- Shows loading state during refresh

## Usage Examples

```javascript
// Changing refresh interval
const intervalSelect = document.getElementById('refresh-interval');
intervalSelect.addEventListener('change', handleIntervalChange);

// Pausing/resuming auto-refresh
const toggleBtn = document.getElementById('refresh-toggle');
toggleBtn.addEventListener('click', handleRefreshToggle);

// Manual refresh
const manualBtn = document.getElementById('refresh-manual');
manualBtn.addEventListener('click', handleManualRefresh);
```
